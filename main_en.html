what is Content-Type in the HTTP request and why do we need it?

The Content Type specifies the type of data being sent to the receiver in order to exchange data properly. This ensures that the data is processed correctly.

For example content types include plain text, html text, and image. The following shows the line codes for the different types of content.

The format is :

Content-Type: text/html\r\n

Content-Type: text/plain\r\n

Content-Type: image/jpg\r\n

Content-Type: image/png\r\n
